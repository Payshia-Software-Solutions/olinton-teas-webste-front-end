# **App Name**: Ceylon Delights

## Core Features:

- Tea Product Catalog: Showcase Ceylon tea varieties with detailed descriptions and pricing.
- Customer Reviews: Display customer testimonials with star ratings and detailed comments.
- Subscription Form: Allow users to subscribe to a newsletter for updates on new trends and exclusive offers.
- Contact Information: Provide detailed contact information including address, phone number, and email.

## Style Guidelines:

- Primary color: Deep forest green (#1A5238) to evoke the lush tea gardens of Ceylon.
- Background color: Light, desaturated olive (#F2F5E7) for a natural and calming feel.
- Accent color: Warm gold (#D4A75A) to highlight important elements and add a touch of luxury. This should contrast clearly against the deep forest green.
- Headline font: 'Playfair', a serif, for an elegant, fashionable, high-end feel; use 'PT Sans', a sans-serif, for body text
- Use handcrafted, detailed icons of tea leaves, cups, and related elements in a gold hue.
- Arrange the homepage into clearly defined sections to improve readability.
- Subtle fade-in effects on images and text as the user scrolls for a smooth user experience.
